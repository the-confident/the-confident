How it works

You create as many issues as you want for us to work on, and place them in the order you want us to work on them.

We work at a rate of maximum 2 open issues at any one time.

Once an issue is passed over for you to test, we stop working until you have it tested. If it passes your testing, we work on the next issue; if not, we continue working on the same issue.

The more responsive you are in testing our work, the more issues we will be working on each month.

What's covered?

We will work on any issue you want us to as long as it's either:

  • Something that exists on your current website
  • Small (new) features that can easily be described in a ticket

We will not work on:

  • Large new features - if we need to have a call about it, it's considered large. We can quote for this work separately.
  • Updates to modules/themes. These updates fall under "hosting" rather than support.
